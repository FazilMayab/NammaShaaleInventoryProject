# Namma Shaale Inventory Project

A dedicated Android application designed to streamline and digitize inventory management for schools. This app helps staff track assets, equipment, and stock levels efficiently, ensuring better accountability and resource management.

## 🚀 Features
* **Inventory Tracking:** Real-time monitoring of school assets and stock.
* **Category Management:** Organize items by department or type.
* **Modern UI:** Built with Jetpack Compose for a smooth user experience.
* **Local Database:** Fast and reliable data storage using Room/SQLite.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **Framework:** Jetpack Compose
* **Database:** Room / SQLite
* **Dependency Management:** Gradle (KTS)

## 📲 Installation
1. Download the `app-debug.apk` file.
2. Install the APK on an Android device (Enable "Install from Unknown Sources" if prompted).
3. Open the app and start managing your inventory!
